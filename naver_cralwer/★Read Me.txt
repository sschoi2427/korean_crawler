★뉴스기사크롤링===============================

1. 네이버 홈페이지에 들어간다

2. 키워드를 검색한다.

3. 기간입력

4. 2페이지로 이동한다

5. URL복사

6. 페이지 수 계산한다 : (뉴스건수/10) >> 소수점이하 무시
>> ex) (2253)/10 = 225.3 >> 225page

7. 스파이더 프로그램 RUN 

8. 2페이지에서 COPY한 URL붙여넣기

9. 6번에서 계산한 페이지 수를 최대값으로 생각해서 입력

★뉴스댓글크롤링===============================

1. 뉴스기사크롤링의 1번~9번까지 동일

2. 댓글 수 입력
  > 기본 값 : 1000개 정도로 생각해서 댓글 수 입력
  > 최대 값이 클수록 시간이 많이 걸림.
